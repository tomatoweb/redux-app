# Redux Toolkit (RTK) with API, JSON-SERVER and Axios

### This is a Redux App example ready to use 

### Redux is a State Management Library for JavaScript applications. 

### It helps manage GLOBALLY the state of an application in a predictable way, making it easier to debug and test.
___
### Topics Covered:

* Provider

* store

* reducers

* actions

* dispatch

* json-server

* Axios
___
### 1. Install Redux
npm i -s redux react-redux @reduxjs/toolkit @redux-devtools/extension

### 2. Install Redux DevTools Chrome extension

### 3. Install JSON-SERVER for the fake REST API calls to db.json

```
npm i -g json-server
```
```
npx json-server --watch db.json
````
#### API's will be running on: 

http://localhost:3000/posts 

http://localhost:3000/users

### 4. Serve frontend
```
npm start
```
